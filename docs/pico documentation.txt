- Wireless LAN
- Bluetooth 5.2
- USB for 1.8-5.5V power

To add the Pico to Arduino IDE
- Add the following to the "Additional Boards Manager URLs" bar under File/Preferences:
- https://github.com/earlephilhower/arduino-pico/releases/download/global/package_rp2040_index.json
- Under Tools/Boards/Boards Manager install Raspberry Pi Pico/RP2040/RP2350 (search for 2040)
